NASA IMS Bearing Dataset - Reconstructed Samples
==================================================

These files are RECONSTRUCTED from the published statistical
characteristics documented in the official NASA IMS dataset.

Original Dataset:
  Source: NASA Prognostics Data Repository
  URL: https://data.nasa.gov/dataset/ims-bearings
  Download: https://data.nasa.gov/docs/legacy/IMS.zip
  Size: ~6 GB (full dataset)

Reference Paper:
  H. Qiu, J. Lee, J. Lin, G. Yu, 'Wavelet filter-based
  weak signature detection method and its application on
  roller bearing prognostics', Journal of Sound and
  Vibration, 289(4-5), pp. 1066-1090, 2006.

Experimental Setup:
  - 4x Rexnord ZA-2115 bearings on a loaded shaft
  - 2000 RPM constant speed, 6000 lbs radial load
  - PCB 353B33 ICP accelerometers at 20 kHz
  - 2nd test: Bearing 1 outer race failure

To obtain the ORIGINAL raw data files:
  1. Visit https://data.nasa.gov/dataset/ims-bearings
  2. Download IMS.zip (~6 GB)
  3. Extract 2nd_test/ directory
